## Installation

'Armadillo' for matrices and vectors
'SFML' to render the environment

## Usage
compile using a gcc compiler as 
'g++ NEAT_Algorithm.cpp -o NEAT -larmadilllo -lsfml-window -lsfml-graphics -lsfml-system'
